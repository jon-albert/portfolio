NHL Showcase

TELUS

UI/UX Design

About this project

NHL Showcase is a hub created for all TELUS Optik users to get regionally specific team information and access to recording features quickly and easily for the start of the 2019-20 NHL season. Content would be delivered based on either geo-location or user preference between all regions covered by TELUS Optik services (Vancouver, Calgary, Edmonton).

My Contributions

Worked closely with TELUS to create user-journeys, wireframes and ultimately the final UI with the help of the lead designer of the project to ensure an experience that would reflect TELUS branding and user guidelines.
